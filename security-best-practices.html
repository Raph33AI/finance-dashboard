1. Account Security

Security Checklist

Enable Two-Factor Authentication

Add an extra layer of security beyond your password.

  • Go to Settings > Security
  • Click "Enable 2FA"
  • Scan QR code with authenticator app (Google Authenticator, Authy)
  • Save backup codes in a secure location

Security Impact: 2FA reduces account takeover risk by 99.9%.

Use a Strong, Unique Password

Your password should be:

  • At least 12 characters long
  • Mix of uppercase, lowercase, numbers, symbols
  • Not used on any other website
  • Not based on personal information
Password Strength Examples

Weak: password123

Medium: MyP@ssw0rd2024

Strong: T7$kL@9mQz!pR4vN

Review Active Sessions

Regularly check logged-in devices:

  • Navigate to Settings > Active Sessions
  • Review device names and locations
  • Revoke access to unrecognized devices

Monitor Login Activity

We email you on every new login from an unrecognized device or location.

Suspicious Activity? If you receive a login notification you don't recognize, immediately change your password and contact support.

3. Two-Factor Authentication

2FA Methods Compared

Method Security Level Convenience Recommendation
Hardware Key (YubiKey) Best for high-value accounts
Authenticator App Recommended for most users
SMS/Text Message Better than nothing, but vulnerable to SIM swapping
Email Code Least secure, only if no alternative

Avoid SMS 2FA for Financial Accounts: SMS can be intercepted via SIM swapping attacks. Use an authenticator app or hardware key instead.

Backup Codes

When enabling 2FA, you'll receive 10 one-time backup codes.

Storage Tips:

  • Print and store in a safe
  • Save in password manager's secure notes
  • Never store in email or cloud without encryption
  • Each code can only be used once

5. Phishing Protection

How to Identify Phishing Attempts

Red Flags:

  • Urgent language: "Your account will be closed in 24h!"
  • Suspicious sender: Check email address (not just display name)
  • Generic greeting: "Dear Customer" instead of your name
  • Spelling errors: Professional companies proofread
  • Unexpected attachments: We never send .exe or .zip files
  • Requests for passwords: We NEVER ask for your password via email

What to Do If You Suspect Phishing

1

Do NOT Click Links or Download Attachments

Immediately delete the email without interacting with it.

2

Forward to Our Security Team

Send the suspicious email to security@alphavault-ai.com

3

Verify Directly

If unsure, log in to your account directly (type URL, don't click links) and check for notifications.

Domain Verification: Our official domain is alphavault-ai.com. Be wary of similar-looking domains like alphavault-ai.net or alpha-vault.com.
