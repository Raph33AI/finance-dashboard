1. Authentication

Getting Your API Key

1

Generate API Key

Navigate to Settings > API Access and click "Generate New Key".

Security: Your API key is shown only once. Store it securely (use environment variables).

2

Authentication Method

Include your API key in the Authorization header of every request:

cURL Example 
curl -X GET "https://api.alphavault-ai.com/v1/stocks/AAPL" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"

Base URL

Production Endpoint

https://api.alphavault-ai.com/v1/

4. Code Examples

Python

Python - Get Stock Analysis 
import requests

API_KEY = "your_api_key_here"
BASE_URL = "https://api.alphavault-ai.com/v1"

headers = {
    "Authorization": f"Bearer {API_KEY}",
    "Content-Type": "application/json"
}

def get_stock_analysis(symbol):
    url = f"{BASE_URL}/stocks/{symbol}/analysis"
    params = {"timeframe": "1M"}
    
    response = requests.get(url, headers=headers, params=params)
    
    if response.status_code == 200:
        return response.json()
    else:
        raise Exception(f"Error: {response.status_code}")

# Usage
analysis = get_stock_analysis("TSLA")
print(f"Prediction: {analysis['prediction']['direction']}")
print(f"Confidence: {analysis['prediction']['confidence'] * 100}%")

JavaScript (Node.js)

JavaScript - Create Portfolio 
const axios = require('axios');

const API_KEY = 'your_api_key_here';
const BASE_URL = 'https://api.alphavault-ai.com/v1';

const createPortfolio = async (portfolioData) => {
  try {
    const response = await axios.post(
      `${BASE_URL}/portfolios`,
      portfolioData,
      {
        headers: {
          'Authorization': `Bearer ${API_KEY}`,
          'Content-Type': 'application/json'
        }
      }
    );
    
    return response.data;
  } catch (error) {
    console.error('Error:', error.response.data);
    throw error;
  }
};

// Usage
const portfolio = {
  name: "My Tech Portfolio",
  positions: [
    { symbol: "AAPL", quantity: 10, purchasePrice: 180.50 },
    { symbol: "MSFT", quantity: 5, purchasePrice: 385.20 }
  ]
};

createPortfolio(portfolio)
  .then(result => console.log('Portfolio created:', result))
  .catch(err => console.error(err));

5. Webhooks

Receive real-time notifications when events occur.

Supported Events

price.alert

Stock reaches target price

news.breaking

Major news for watched stocks

portfolio.rebalance

Rebalancing recommended

risk.threshold

Portfolio risk exceeds limits

Webhook Payload Example

POST to your endpoint 
{
  "event": "price.alert",
  "timestamp": "2025-01-15T14:30:00Z",
  "data": {
    "symbol": "TSLA",
    "price": 250.00,
    "targetPrice": 250.00,
    "alertId": "alert_abc123"
  }
}

Security: Webhooks include an X-AlphaVault-Signature header for verification. Validate this signature to ensure authenticity.
