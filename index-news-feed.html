How It Works

Data Flow Architecture

From RSS feeds to AI-powered insights in 4 steps

1

RSS Data Collection

RSSClient fetches articles from multiple financial news sources (CNBC, MarketWatch, Bloomberg, Reuters, FT). Supports 100-200 articles per source with caching for performance.

2

Entity Extraction

EntityDatabase extracts company names, tickers, and countries from article titles. Database contains 300+ companies (AAPL, MSFT, NVDA...) and 50+ countries.

3

Sentiment Analysis

SentimentAnalyzer classifies articles as Positive/Neutral/Negative using keyword matching. Scores range from -100 to +100 with confidence levels.

4

Dashboard Rendering

Chart.js visualizes data with gauges, timelines, sparklines, and bars. Real-time updates with smooth animations and interactive tooltips.
