Budget Dashboard

Complete Financial Tracking

8 Highcharts visualizations with Cloud Sync, auto-save, and inflation-adjusted analysis

Dynamic Timeline

Add/remove months dynamically. Auto-adjusts calculations for unlimited time horizons. Tracks income, expenses, savings, and investments with monthly granularity.

AI ROI Calculator

Customizable annual yield (default 8%). Computes monthly gains with compound interest, cumulated portfolio value, and ROI % automatically.

8 Highcharts Visualizations

Income vs Expenses, Monthly Breakdowns (pie), Cumulated Savings (area), Portfolio Evolution, ROI Evolution, Budget 50/30/20 rule, Multi-currency converter.

Inflation-Adjusted Mode

Toggle to visualize real value curves adjusted for inflation (customizable rate, default 2.5%). See nominal vs real savings and portfolio growth side-by-side.

Cloud Sync (Firestore)

Real-time synchronization via Cloudflare Workers + Firestore. Auto-save every 2 seconds (debounced). LocalStorage fallback ensures data safety.

Bulk Editing

Apply values to multiple months at once. Select category (salary, rent, investment), date range, and amount. Instant batch updates with auto-recalculation.

Export/Import JSON

Backup entire timeline to JSON file. Import data from previous exports. Portable format for migrations and auditing.

Collapsible Data Table

Toggle visibility of the detailed data table. State saved in localStorage. Clean interface focusing on charts or detailed row-by-row editing.

Investment Analytics

Professional Portfolio Optimization

Multi-allocation system with AI-powered Efficient Frontier and 14 advanced risk metrics

Multi-Allocations

Create unlimited allocation simulations. Link to Budget Dashboard simulations. Toggle between % and EUR modes. Compare up to 4 allocations side-by-side.

Efficient Frontier

Visualize 100 optimal risk/return portfolios. Plot current allocation vs efficient frontier. Identify under-optimized positions with Sharpe Ratio analysis.

AI Optimized Strategies

4 differentiated strategies: Conservative (min volatility), Balanced (max Sharpe), Aggressive (max return), Custom (risk-profile tailored). One-click application.

Historical Backtest

Simulate portfolio performance on historical data. Compare strategies' final values, annualized returns, max drawdowns, and Sharpe ratios side-by-side.

14 Risk Metrics

Sharpe, Sortino, Calmar, VaR, CVaR, Max Drawdown, Volatility, Win Rate, Profit Factor, Rolling metrics, Alpha/Beta, Correlation Matrix, and more.

Diversification Score

0-100 score analyzing: number of assets, asset class diversity, allocation balance, concentration risk, and estimated correlation. Actionable insights included.

14 Highcharts Visualizations

Portfolio Evolution, Monthly Returns, Drawdown, Rolling Volatility, Returns Distribution, VaR, Correlation Heatmap, Rolling Sharpe/Sortino, Alpha/Beta, Calmar, Efficient Frontier, Backtest.

AI Recommendations

Contextual insights based on risk profile, diversification score, Sharpe Ratio, and concentration. Prioritized actions (high/medium/low) with actionable buttons.
